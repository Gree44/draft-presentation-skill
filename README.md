# draft-presentation — Claude Code Skill

A Claude Code skill that walks you through creating a structured presentation draft slide by slide, then hands it off to `marp-slides` for generation.

## What it does

Instead of asking Claude to generate a full presentation in one shot (which produces generic, incoherent results), this skill runs an interactive process:

1. **Upfront Q&A** — Collects topic, audience, duration, language, assessment criteria, research material, and theme in one message
2. **Slide overview** — Proposes a numbered slide list for your approval before writing anything
3. **Slide-by-slide drafting** — Works through each slide iteratively, incorporating your feedback before moving on
4. **Handoff** — Writes `presentation_draft.md` and automatically invokes `marp-slides` to generate the final `presentation.md`

The draft format includes a Kernaussage (core message), visual layout description, full speaker notes written as spoken prose, and transition sentences — not bullet points.

## Recommended workflow

1. **Research first** — Run `/deep-research` on your topic before starting the draft. Pass the research output as your source material in Phase 1 of this skill. This ensures factual accuracy and gives Claude the depth needed to write complete speaker notes.

2. **Draft** — Run `/draft-presentation` and work through the slides interactively.

3. **Generate** — The skill invokes `/marp-slides` automatically at the end, or you can run it manually on the saved `presentation_draft.md`.

## Requirements

### Claude Code
- [Claude Code](https://claude.ai/code) CLI or VS Code extension

### marp-slides skill
This skill invokes `marp-slides` at the end of the process. You need that skill installed too — copy `.claude/skills/marp-slides/` from the [marp-slides repository](https://github.com/anthropics/claude-code-skills) or your own installation into the same `.claude/skills/` directory alongside this skill.

### MARP (for rendering the output)
- VS Code extension: [Marp for VS Code](https://marketplace.visualstudio.com/items?itemName=marp-team.marp-vscode)
- Enable in VS Code settings:
  ```json
  "markdown.marp.enableHtml": true
  ```
- CLI export (optional):
  ```bash
  npx @marp-team/marp-cli presentation.md --pdf --allow-local-files
  ```

## Installation

Copy the skill into your Claude skills directory:

**Global (available in all projects):**
```bash
cp -r .claude/skills/draft-presentation ~/.claude/skills/
```

**Project-level (only in current project):**
```bash
cp -r .claude/skills/draft-presentation /your/project/.claude/skills/
```

Then invoke it in Claude Code:
```
/draft-presentation
```

## Output

The skill produces two files in your working directory:
- `presentation_draft.md` — structured draft with speaker notes, layout descriptions, and design hints
- `presentation.md` — the final MARP slide deck (generated by `marp-slides`)
